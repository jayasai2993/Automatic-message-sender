# Automatic-message-sender
automatic message sending assistant which is done with python <br/> <br/>
Uses and Limitations: <br/> 
Uses: Automates scheduled WhatsApp messages, which can be useful for reminders, birthday wishes, etc. <br/> <br/>
Limitations: <br/> 
Requires the WhatsApp Web session to be active, so the user must scan the QR code on WhatsApp Web when prompted. <br/>
The pywhatkit module relies on the browser and internet connection, and it's not designed for high-volume messaging.
